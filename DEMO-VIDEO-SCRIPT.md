## Recording Tips

### Preparation
1. **Clear browser cache** to show fresh experience
2. **Prepare sample answers** for interview questions
3. **Test all flows** before recording
4. **Use high-quality screen recording** (1080p minimum)
5. **Record audio separately** for better quality

### During Recording
1. **Slow down cursor movements** for clarity
2. **Pause briefly** after each action
3. **Use zoom/highlight** for important elements
4. **Keep narration pace steady** (not too fast)
5. **Show loading states** to demonstrate real-time AI

### Editing
1. **Add transitions** between sections
2. **Highlight UI elements** with circles/arrows
3. **Add text overlays** for key points
4. **Include background music** (subtle, non-distracting)
5. **Add captions** for accessibility

### Technical Setup
- **Screen Resolution**: 1920x1080
- **Frame Rate**: 30 or 60 fps
- **Audio**: Clear microphone, no background noise
- **Browser**: Chrome (latest version)
- **Extensions**: Disable ad blockers, enable React DevTools (optional)
